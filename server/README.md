# Backend

Серверная часть приложения «Custom Hooks».  
Описание проекта доступно в [главном README](../README.md).

## Назначение

Сервер предоставляет собой REST API для обработки запросов от клиента.  
Обрабатывает HTTP-запросы и возвращает данные.

Основные задачи:

- обработка успешных запросов
- имитация ошибок
- задержка ответа для тестирования состояния загрузки

## API

- **GET /ping** — проверка подключения к серверу
- **GET /success** — успешный ответ от сервера
- **GET /error** — имитация ошибки 500
- **GET /loading** — имитация задержки с успешным ответом спустя 5 секунд

## Архитектура

Простой REST API для обработки запросов от клиента.  
Все данные обрабатываются в памяти и не сохраняются после перезапуска сервера.

## Технологии

- Node.js
- Express
- CORS
- REST API
